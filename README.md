# nami-ui-react

component library for react

## 安装

`npm install nami-ui-react --save`

`yarn add nami-ui-react`

## 示例

```js
import { Button } from 'nami-ui-react';
const App = () => (
    <Button btnType="primary" disabled>PRESS ME</Button>
);
```

引入样式

```js
import 'nami-ui-react/dist/index.css';
```
